<objective>
Repair the structured-hypothesis DAG semantics identified in UAT by separating
round-level diversity reporting from node-level metrics, removing the
"unknown vs zero diversity" ambiguity, and wiring later rounds to create true
non-root DAG nodes through SelectParentsService.

Purpose: Phase 26's topology layer should model actual path structure and
meaningful diversity information, not just produce green tests around copied
entropy values. This plan closes the semantic gap before Phase 27 depends on
the DAG for cross-branch reasoning.

Output: clarified diversity contract, parent-aware later-round DAG creation,
and integration tests that prove node metrics and topology are meaningful.
</objective>
