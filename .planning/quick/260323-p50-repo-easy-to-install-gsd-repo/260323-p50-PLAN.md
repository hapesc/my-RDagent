FIRST, update `pyproject.toml` to add a `lint` optional-dependencies group so that ruff is installable via uv:
```
[project.optional-dependencies]
test = [
    "pytest>=7.4.0",
    "hypothesis>=6.0.0",
    "import-linter>=2.3,<3.0",
]
lint = [
    "ruff>=0.3",
]
```

Create `.github/workflows/ci.yml` with:
- Trigger on push to main/master and on all pull requests
- Matrix: Python 3.11 + 3.12, os: ubuntu-latest + macos-latest (4 jobs)
- Steps: checkout, install uv (astral-sh/setup-uv@v4), `uv sync --extra test --extra lint`, then `make lint` and `make test`
- IMPORTANT: the sync command MUST include both `--extra test` AND `--extra lint` so that both pytest and ruff are available
- Set a 10-minute timeout per job (matching GSD pattern)
- Cache uv dependencies using the built-in uv cache from setup-uv
