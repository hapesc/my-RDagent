### Fields Added to NodeRecord
1. visits: int = 0
   - Counts number of times update_stats() called
   - Defaults to 0 for backward compatibility

2. total_value: float = 0.0
   - Accumulates all rewards passed to update_stats()
   - Defaults to 0.0 for backward compatibility

3. avg_value: float = 0.0
   - Average reward = total_value / visits
   - Defaults to 0.0 for backward compatibility

### Method Added to NodeRecord
update_stats(self, reward: float) -> None:
    - Increments visits by 1
    - Adds reward to total_value
    - Computes avg_value = total_value / visits
    - Handles edge cases: zero rewards, negative rewards, large numbers
